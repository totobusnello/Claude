HERDR Cockpit

herdr 0.7.4 · 17 workspaces curados · 6 plugins · prefix Ctrl+B
Toda tecla é Ctrl+B → solta → a tecla.

Não é acorde simultâneo. prefix+shift+j = aperta Ctrl+B, solta, aperta Shift+J.

Esqueceu algum? Ctrl+B solta Shift+/ (que é o ?) lista todos ao vivo. Sai com Esc.

Ou nem use teclado: o herdr é mouse-native — clique em pane/tab/workspace, arraste bordas, botão direito abre menu.

O fluxo do dia

  1. herdr no iTerm Conecta na sessão que já está rodando. De qualquer diretório. Nunca rode dentro de um pane do herdr.
  2. prefix+shift+o — monta os 17 workspaces curados Lê a allowlist ~/Claude/.herdr/spaces. Projeto faltando? Adicione o path lá.
  3. prefix+shift+j — pula pro projeto Digita "viridi", Enter. É o atalho mais útil do dia.
  4. cw no pane da esquerda Briefing do projeto + Claude já com o contexto. wb mostra só o briefing.
  5. prefix+shift+i — revisa o diff quando o agent terminar
  6. prefix+q — sai, e tudo continua rodando Pode fechar o iTerm e o notebook. Volta com herdr.

Dentro do reviewr o loop de review

Abre com prefix+shift+i quando o agent terminar de escrever código. Abre sozinho quando um worktree é criado.

v depois j kseleciona linhas (ou arrasta com o mouse)
ccomenta na seleção
e / dedita / apaga o comentário
n / Npróximo / anterior comentário
llista todos
smanda os comentários pro input do agent
O que o reviewr é — e o que não é. Ele não substitui /kimi:review, /glm:review, /grok:review. Essas valem pelo que são: famílias de treino distintas te dando divergência real. O reviewr substitui o transporte do feedback de volta pro agent — o copiar-e-colar.

Em ~/Claude/scripts/

A regra que vale ouro. Mexeu no config.toml à mão? Rode herdr-doctor.sh --save depois. Senão o próximo herdr update restaura o snapshot antigo e sua mudança evapora — o doctor é um sistema imunológico, ele rejeita o que não conhece.
Atualizar o herdr: é brew upgrade herdr, não herdr update — o self-update recusa em instalação Homebrew. Depois, restart: launchctl kickstart -k gui/$(id -u)/dev.herdr.server.
