## Workspace & Sync

### Directory Structure
```
~/Claude/                          (source of truth)
├── skills/          → 91 directories, 319 SKILL.md files
├── agents/          → 22 agent categories
├── commands/        → 218 command files
├── prompts/         → 41 prompt templates
├── templates-powerpoint/  → 93 PowerPoint templates (submodule)
├── Projetos/              → 7 project repos + agent-orchestrator (submodules)
├── scripts/               → Sync and automation scripts
├── agent-orchestrator.yaml → AO configuration
├── CLAUDE.md              → Project instructions
├── CATALOG.md             → Detailed skills/agents catalog
├── INDEX.md               → This file
└── _retired/              → Deprecated skills/commands
```

### Auto-Sync Configuration
All extensions auto-sync to `~/.claude/` every night via launchd (23h interval).

Manual sync:
```bash
~/Claude/scripts/sync-all-to-home.sh
```

Deployment targets:
- `~/.claude/skills/` ← synced from `~/Claude/skills/`
- `~/.claude/agents/` ← synced from `~/Claude/agents/`
- `~/.claude/commands/` ← synced from `~/Claude/commands/`
- `~/.claude/prompts/` ← synced from `~/Claude/prompts/`
