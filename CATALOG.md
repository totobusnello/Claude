## Installation & Sync

All skills, agents, commands, and prompts are managed in the `~/Claude/` workspace and auto-synced to `~/.claude/` every night.

### Manual Sync
```bash
~/Claude/scripts/sync-all-to-home.sh
```

### Usage in Claude Code
Skills are activated automatically by keywords or used explicitly:
```
Use the autonomous-dev skill to build this feature
Use the tob-semgrep skill to scan for security issues
Use the ui-ux-pro-max skill to design the interface
```

Agents are invoked explicitly:
```
Use the 02-language-specialists agent for TypeScript
Use the 04-quality-security agent for security review
Use ralph-agents for long-running background work
```
