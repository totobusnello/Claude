Architecture

Five layers, one SQLite file.

  1. Ingest — router auto-detects entity files, markdown, or graphify input. Privacy filter applies 13 redaction patterns pre-storage.
  2. Store — chunks land in SQLite with FTS5 index plus 3072-d Gemini vector via sqlite-vec. Typed retention: feedback / person never decay; lesson 180d; decision 365d.
  3. Retrieve — query runs in parallel through FTS5 BM25 and Gemini semantic. RRF fusion (k=60) merges. Language-aware weights tilt dense up on PT queries.
  4. Rank — salience (recency × pain × importance) composes additively with section_boost and temporal boost. Shadow mode is the default.
  5. Answer — CLI, MCP, and HTTP surfaces with citation footers, anti-hallucination guard, and phase-broken-down latency budget.
