# Code of Conduct

This project adopts the **[Contributor Covenant version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct.

The full text is available at the link above. By participating in this project (contributors, maintainers, reviewers, issue reporters, and other community members), you agree to abide by it.

## Summary

We commit to providing a welcoming, harassment-free experience for everyone. We expect:

- **Respectful communication** — disagreement on technical decisions is welcome; personal attacks are not.
- **Constructive feedback** — focus on the work, not the person.
- **Inclusion** — anyone willing to contribute, regardless of background, is welcome.
- **Privacy** — respect others' personal information; do not publish private communications without consent.

## Enforcement

Violations may be reported to the maintainers via:

1. **GitHub Security Advisory** (for sensitive reports): https://github.com/totobusnello/memoria-nox/security/advisories
2. **GitHub Issues** (for non-sensitive reports): tag with `code-of-conduct`
3. **Email**: `lab@nuvini.com.br` — subject: `[CoC] memoria-nox — <brief description>`

All reports are reviewed and investigated. Maintainers are obligated to respect the privacy and security of reporters.

## Scope

This Code of Conduct applies within all project spaces (issues, PRs, discussions, code reviews) and when an individual is officially representing the project in public spaces (conferences, blog posts, social media).
