## Error paths the script handles

| Condition | Behaviour |
|---|---|
| VPS unreachable (scp timeout 15s) | Exits with clear error + monitor commands |
| CSV missing critical file | Exits with error + suggests checking tmux |
| nDCG value outside [0, 1] | Validation abort — possible unit error flagged |
| Expected system row missing (bm25 or e5) | Validation abort — CSV system name mismatch flagged |
| Fewer than 50 queries | Validation warning + abort — partial run |
| `\label{tab:beir}` not found in .tex | Exits with error — manual patch needed |
| LaTeX compile error | Exits with last 30 lines of pdflatex stdout |
| CSV absent but JSONLs present | Auto-aggregates from per-query JSONL, writes synthetic CSV |
