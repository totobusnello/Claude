## "Nox" Iconography

The name "Nox" (Latin: night) anchors the visual language. The iconography should
evoke:
- Memory retrieval — light emerging from darkness
- Precision — structured, geometric, not decorative
- Ownership — solid, contained, not cloud/network imagery

Avoid: brain illustrations, generic neural-network node graphs, cloud icons.
Preferred: single-point light source, layered depth, minimal geometry.

---

## Usage Guidelines

- Do not stretch, rotate, or recolor the logo without permission.
- Minimum clear space: 1× the height of the "n" in nox-mem on all sides.
- On backgrounds with insufficient contrast, use the dark or light variant
  (not a color-adjusted version of the primary).
- For embargoed pre-launch press use, contact lab@nuvini.com.br to request
  draft assets.
