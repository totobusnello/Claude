# What did you build with nox-mem? Share here

This thread is for showing what you've built using nox-mem — agents, integrations, custom pipelines, anything that runs on top of it.

## What to share

No format required. Some prompts to get started:

**Agents and workflows**
- What agent did you build? What does it remember across sessions?
- Which MCP tools do you use most (`nox_mem_search`, `kg_build`, `reflect`...)?
- How are you calling nox-mem — via MCP server, HTTP API, or direct CLI?

**Performance in your setup**
- Corpus size (chunks, entities)?
- Search latency at your scale?
- VPS or local? Specs?

**Patterns and discoveries**
- A retrieval pattern you found that works particularly well
- How you structure your entity files (see `docs/USE-CASES.md` for examples)
- A use case not covered in the docs that you've made work

**Integrations**
- nox-mem + LangChain, LlamaIndex, CrewAI, AutoGen, or a custom framework
- nox-mem as memory for a specific model host (Claude, GPT-4, local Llama)
- Any wrapper, adapter, or tooling you've built around the CLI or HTTP API

## Format

A few sentences describing what you built is enough. Screenshots, links, and benchmark numbers are a bonus. No minimum, no maximum.

Standout entries will get featured in the README or mentioned on social. Community builds are the best signal for what the library is actually useful for — that directly shapes roadmap priority.

Looking forward to seeing what you've made.
