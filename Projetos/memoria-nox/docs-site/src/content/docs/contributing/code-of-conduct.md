This project follows the [Contributor Covenant](https://www.contributor-covenant.org/) v2.1.

## Summary

Be respectful, be constructive, be kind. We are building tools for human memory augmentation — the community should reflect the same care we put into the product.

**Not acceptable:** harassment, discrimination, personal attacks, publishing others' private information.

**Encouraged:** constructive critique, inclusive language, welcoming newcomers, attributing ideas and credit.

## Reporting

Report conduct issues to: conduct@nuvini.com.br

All reports are reviewed confidentially. We follow the enforcement guidelines in the Contributor Covenant.
