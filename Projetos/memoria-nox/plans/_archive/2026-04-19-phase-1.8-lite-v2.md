**Envelope schema (TypeScript):**
```ts
interface DispatchEnvelope {
  intent: "research" | "valuation" | "contract_review" | "security_audit" | "content_post" | "dev_task" | "other";
  subject: string;        // até 200 chars
  params: Record<string, string | number>;  // structured hints
  quoted_external?: string;  // texto externo não confiável; será wrapped em <untrusted>
  urgency: "normal" | "high";
  thread_id?: string;     // Discord thread_id se existir
  context_ref?: string;   // chunk_id:N ou URL
}
```

**Server-side (gateway) adiciona:**

- `subject.length <= 200`
- `quoted_external.length <= 8192`
- `intent` in enum
- JSON Schema validate antes de persistir
